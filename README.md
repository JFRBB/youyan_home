



启动服务：npm run start

打包编译：npm run build


index.js:  入口文件 ---> 配置路由

api文件夹：封装的api请求、统一api路径

common文件夹： 共用方法和常用的验证判断

components文件夹：公共组件

containers文件夹：项目页面 --- 文件命名首字母大写 最外层容器以id命名(不同文件的id也别重名，避免样式污染)

static/img文件夹：放置静态图片

main.js/newPage.js: 路由的跳转方法、redux用法参考、API的用法范例（勿删）

styles/base.css: 全局样式
